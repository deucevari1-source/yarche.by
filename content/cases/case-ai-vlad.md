# ai-Влад — AI-ассистент для сайта агентства

**Клиент:** агентство ЯРЧЕ (внутренний проект) · **Год:** 2026

AI-сотрудник, который работает на сайте 24/7 вместо менеджера: квалифицирует входящие лиды, отвечает на вопросы об услугах и передаёт готовые заявки в Telegram. Не очередной FAQ-бот — ведёт структурированный диалог и подключается к сайту одной строкой кода.

## Стек

| Слой | Технология |
|---|---|
| Виджет | Vanilla JS, Shadow DOM (~15 KB) |
| API | Node.js + Hono framework |
| Модель | Claude Haiku (Anthropic API) |
| Стриминг | Server-Sent Events |
| Инфраструктура | PM2 + nginx + Let's Encrypt, Docker |

## Что внутри

- **Подключение одной строкой:** `<script src="/widget.js" data-api="…" data-name="…">` — встраивается на любой сайт
- **Shadow DOM** — стили не конфликтуют с сайтом, виджет полностью изолирован
- **Стриминг ответов** в реальном времени, побуквенно — пользователь не ждёт «крутилку»
- **Адаптивный UI:** круглая кнопка на десктопе, пилюля на мобиле, отдельные фиксы под iOS Safari
- **Сессия в sessionStorage** на 1 час — история диалога переживает переходы между страницами
- **Квалификация лидов**: бот узнаёт что за бизнес, какие задачи и проблемы, и сам предлагает оставить контакт
- **Заявки уходят в Telegram** в том же формате, что и с форм сайта
- **Стоимость диалога ~$0.001–0.003** — Claude Haiku хватает для консультаций
