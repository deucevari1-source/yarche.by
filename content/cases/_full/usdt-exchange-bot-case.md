# USDT Exchange Bot

**Telegram-бот для обменного сервиса криптовалют в Минске**

---

## Клиент

Обменный сервис криптовалют, Минск

## Задача

Автоматизировать приём заявок на обмен USDT, внедрить реферальную программу для органического роста клиентской базы, дать владельцу удобный инструмент управления прямо из Telegram.

## Что сделали

Разработали Telegram-бота, который полностью закрывает цикл работы обменника: от первого касания клиента до завершения сделки и начисления реферальных бонусов.

### Клиентская часть

Бот стал единой точкой входа — оформление заявки занимает 30 секунд: выбрал направление, ввёл сумму, указал удобное время. Клиент видит статус своей заявки в реальном времени прямо на главном экране бота.

### Реферальная система

Работает полностью автоматически: каждый пользователь получает персональную ссылку, при сделке реферала бонус начисляется без участия оператора. Пользователи видят свой баланс, количество приглашённых и заработок в любой момент.

### Админ-панель

Полноценное управление внутри Telegram: статистика по объёмам и прибыли, управление заявками в одно касание, гибкая настройка комиссии, рассылка объявлений всем пользователям.

## Функционал

| Модуль | Описание |
|---|---|
| Заявки | Покупка/продажа USDT с выбором суммы, времени и комментария |
| Реферальная система | Уникальная ссылка, автоматическое начисление бонусов, отображение пригласившего |
| Личный кабинет | Баланс бонусов в USDT, объём сделок, количество рефералов |
| Админ-панель | Статистика, управление заявками, топ рефереров |
| Настройка комиссии | Изменение на лету (1–3%), сразу отображается у пользователей |
| Рассылка | Объявления всем пользователям через команду или кнопку с предпросмотром |

## Стек технологий

Python · python-telegram-bot · SQLite · Ubuntu · systemd

## Результат

- Время обработки заявки сократилось с 5–10 минут переписки до мгновенного уведомления
- Реферальная программа запущена с первого дня без дополнительных затрат
- Владелец управляет бизнесом с телефона — без веб-панелей и дашбордов
- Бот развёрнут на VPS, работает 24/7 с автоматическим перезапуском
