# avtoschool.by — каталог автошкол и инструкторов Минска

**Сайт:** [https://avtoschool.by](https://avtoschool.by)
**Тип:** Контент-сервис, независимый агрегатор
**Год:** 2025–2026

## О проекте

Независимая платформа выбора автошколы и инструктора по вождению в Минске.
Каталог 100+ школ и инструкторов с реальными отзывами учеников,
рейтингами, ценами по категориям, интерактивной картой и блогом.
Полностью самостоятельная разработка без шаблонов, чужих движков и SaaS-конструкторов.

## Стек

| Слой | Технология |
|---|---|
| Frontend | Next.js 16 (App Router, Turbopack), React 19, Tailwind CSS 4 |
| Backend | Next.js Route Handlers, Prisma 7 ORM |
| База данных | PostgreSQL 16 |
| Аутентификация | JWT (jose) на HttpOnly-cookies, bcryptjs |
| Карта | Leaflet + MapLibre GL |
| Инфраструктура | Docker Compose, nginx, Let's Encrypt, Cloudflare DNS |
| Хостинг | Беларусь, всё в национальной юрисдикции |
| Интеграции | Telegram Bot API (заявки на запись) |

## Функционал

- **Каталог школ** с фильтрами по районам, метро, категориям (A/B/C/D), цене, наличию автодрома, типу коробки, рассрочке
- **Профили инструкторов** с привязкой к школам, опытом, ценой за урок, фото автопарка
- **Система отзывов** с модерацией: рейтинг, текст, защита от спама, лимит на пользователя
- **Внешние рейтинги** — подтягиваются с Яндекс.Карт и Google Maps
- **Интерактивная карта Минска** с пинами школ, привязкой к метро
- **Блог** с поддержкой Markdown в админке
- **Личный кабинет** для учеников и инструкторов
- **Админ-панель** для модерации контента и заявок
- **Telegram-уведомления** обо всех заявках на запись

## Технические решения

### Безопасность
- Rate limiting на критичных эндпоинтах (login, register, enroll, отзывы)
- Magic-bytes валидация загружаемых файлов вместо доверия MIME
- HTML-escaping всех пользовательских данных, попадающих в Telegram
- Транзакционные удаления с каскадами и чисткой связанных сущностей
- Полный аудит JWT и сессий, обязательный `NEXTAUTH_SECRET` без fallback

### Соответствие законодательству РБ
Платформа разработана с учётом требований **Закона Республики Беларусь
от 7 мая 2021 г. № 99-З «О защите персональных данных»**.

- Все данные хранятся **физически в РБ**
- Никакой трансграничной передачи персональных данных
- Подробный пользовательский [документ согласия](https://avtoschool.by/soglasie)
  с реквизитами, перечнем данных, целями, сроками и правами субъекта
- Журналирование согласий с фиксацией версии, IP и user-agent

### SEO и индексация
- Server-side rendering всех страниц через Next.js
- Динамический sitemap.xml со всеми школами, инструкторами, новостями
- Структурированные данные schema.org: `EducationalOrganization`,
  `Person`, `Article`, `AggregateRating`, `PostalAddress`
- Canonical URL, Open Graph, Twitter Card на каждой странице
- Корректные robots.txt и метаданные

### Миграция домена с историей
Базовая площадка изначально жила на новом домене.
В процессе развития переехали на `avtoschool.by` (тематический домен с историей)
с полным сохранением SEO-сигналов:
- 301 редирект со всех URL старого домена на новый с сохранением путей
- Submit `Change of Address` в Google Search Console
- Главное зеркало в Я.Вебмастере
- Прозрачная для пользователей миграция без даунтайма
