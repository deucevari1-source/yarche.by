# Hair Atelier — сайт-визитка и CRM салона красоты

**Год:** 2026

Полностью самописная платформа для салона красоты: публичный сайт-визитка с онлайн-записью и админ-CRM для управления салоном. Прямой конкурент YCLIENTS / DIKIDI, но без подписки и привязки к чужой инфраструктуре — весь код, база и Telegram-боты живут на одном VPS клиента, без посредников и абонентки.

## Что внутри

- **Публичная часть** — каталог услуг, мастера, пошаговая запись, витрина товаров, курсы. Mobile-first с нижним tab-bar и pull-to-refresh
- **Личный кабинет клиента** — JWT по телефону, список записей, ответы на сообщения салона
- **Календарь админки** — недельная сетка по мастерам, 30-минутные слоты, цветовые статусы, инлайн-оплата
- **CRM для клиентов** — история визитов, заметки мастера, предпочтения (аллергии, окрашивание)
- **Финансы** — отчёт по выручке за период с разбивкой по методам оплаты, мастерам и услугам
- **Своя система сообщений клиентам** — автогенерация повторных записей, опросы и акции вместо платных SMS-маркетинга YCLIENTS
- **Telegram-бот** для мастеров и админов — быстрые действия (подтвердить / отменить / закрыть запись) прямо в чате
- **Склонение русских ФИО** — обращения «к мастеру **Анне**» вместо «к Анна»

## Как это выглядит

### Главная для авторизованного клиента

Та страница, на которую попадает клиент после авторизации по телефону. Персональное приветствие со склонением имени, карточка ближайшей записи с её статусом и блок сообщений от салона — клиент может ответить прямо здесь, без перехода в мессенджеры. Минималистичная сетка, фирменный знак Hair Atelier справа.

### Дашборд админки

![Дашборд админки Hair Atelier](/images/cases/hair-atelier/admin-dashboard.webp)

Стартовый экран администратора. Сверху — счётчики: записи на сегодня, ожидающие подтверждения, на текущей неделе, всего в системе. Снизу слева — лента последних записей со статусами, справа — быстрые ссылки на разделы управления (мастера, товары, курсы).

### Журнал записей — недельный календарь по мастерам

![Журнал записей Hair Atelier](/images/cases/hair-atelier/admin-journal.webp)

Главный рабочий экран администратора. Сверху — записи, которые ждут подтверждения (с быстрыми кнопками «подтвердить / отменить»). Снизу — недельная сетка по мастерам с 30-минутными слотами. Цветовое кодирование статусов: подтверждённая запись, ожидание, обед, перерыв. Переключатель день / неделя / месяц.

### Клиентская база

![Клиентская база Hair Atelier](/images/cases/hair-atelier/admin-clients.webp)

CRM по клиентам салона. По каждому клиенту — телефон, количество визитов, сумма потраченного, дата последнего визита. Быстрый поиск по имени или телефону. Клик по строке открывает карточку клиента с историей записей, заметками мастера и предпочтениями (аллергии, окрашивание).

### Управление мастерами и графиками

![Управление мастерами Hair Atelier](/images/cases/hair-atelier/admin-masters-schedule.webp)

Карточки мастеров с их специализацией и количеством записей за всё время. Модальное окно «График работы» — выбор рабочих дней через календарь и индивидуальное время работы по каждой дате. Поддержка нестандартных графиков (отпуск, частичный день, перерыв на обед).
