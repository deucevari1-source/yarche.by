# USDT Exchange Bot — Telegram-бот для обменника криптовалют

**Клиент:** обменный сервис криптовалют, Минск · **Год:** 2025

Telegram-бот, который полностью закрывает цикл работы обменника USDT: от первой заявки клиента до завершения сделки и начисления реферальных бонусов. Владелец управляет бизнесом с телефона — без веб-панелей и дашбордов.

## Стек

| Слой | Технология |
|---|---|
| Бот | Python + python-telegram-bot |
| База данных | SQLite |
| Инфраструктура | Ubuntu + systemd, автоперезапуск, VPS 24/7 |

## Что внутри

- **Оформление заявки за 30 секунд** — выбор направления, сумма, удобное время. Клиент видит статус в реальном времени
- **Реферальная программа** на автомате — у каждого своя ссылка, бонусы начисляются без участия оператора, пользователи видят свой баланс и количество приглашённых
- **Админ-панель внутри Telegram** — статистика по объёмам и прибыли, управление заявками в одно касание, топ рефереров
- **Гибкая настройка комиссии** (1–3%) — меняется на лету, сразу применяется к пользователям
- **Рассылка объявлений** всем пользователям через команду или кнопку с предпросмотром
