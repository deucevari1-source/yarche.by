# china-minsk.by — доставка товаров из Китая в Беларусь под ключ

**Сайт:** [china-minsk.by](https://china-minsk.by) · **Год:** 2025–2026

Промо-сайт логистической компании: карго, официальная доставка, таможенное оформление, выкуп с 1688 и Taobao. Цель — собирать заявки на расчёт стоимости и максимально широко индексироваться в поиске по геозапросам по всем крупным городам Беларуси.

## Стек

| Слой | Технология |
|---|---|
| Frontend | Pure HTML5 / CSS3 / Vanilla JS — без фреймворков и сборщиков |
| Backend для лидов | Python 3, FastAPI |
| Уведомления | Telegram Bot API |
| Хостинг | cPanel (статика) + отдельный VPS для API |

## Что внутри

- **~25 страниц**, заточенных под скорость и SEO
- **16 городских лендингов** — Минск, Брест, Витебск, Гомель, Гродно, Могилёв и другие. Каждый — отдельная страница с уникальной OG-картинкой, метаданными и геолокальным контентом
- **7 страниц услуг** + кастомная 404
- **Двумодальная лид-форма**: «есть ссылка на товар» / «опишу словами» — поля адаптируются под сценарий
- **24 уникальные OG-картинки** под соцсети и мессенджеры
- Заявки уходят в FastAPI и пересылаются менеджерам в Telegram

## Результат

Через две недели после запуска у клиента начались ежедневные звонки по заявкам с сайта.
