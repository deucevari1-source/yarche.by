# avtoschool.by — каталог автошкол и инструкторов Минска

**Сайт:** [avtoschool.by](https://avtoschool.by) · **Год:** 2025–2026

Независимая платформа выбора автошколы и инструктора по вождению в Минске. Каталог 100+ объектов с реальными отзывами учеников, рейтингами, ценами по категориям, интерактивной картой и блогом. Полностью самостоятельная разработка без шаблонов и SaaS-конструкторов.

## Стек

| Слой | Технология |
|---|---|
| Фреймворк | Next.js 16 (App Router), React 19, TypeScript |
| Стили | Tailwind CSS 4 |
| Backend | Next.js Route Handlers + Prisma 7 |
| База данных | PostgreSQL 16 |
| Карта | Leaflet + MapLibre GL |
| Хостинг | Беларусь, всё в национальной юрисдикции |

## Что внутри

- **Каталог 100+ школ и инструкторов** с фильтрами по районам, метро, категориям (A/B/C/D), цене, наличию автодрома, типу коробки и рассрочке
- **Профили инструкторов** с привязкой к школам, опытом, ценой за урок, фото автопарка
- **Система отзывов** с модерацией, защитой от спама и подгрузкой рейтингов с Яндекс.Карт и Google Maps
- **Интерактивная карта Минска** с пинами школ и привязкой к метро
- **Личный кабинет** для учеников и инструкторов, **админ-панель** для модерации
- **Уведомления в Telegram** обо всех заявках на запись
- **Соответствие закону РБ № 99-З** «О защите персональных данных» — все данные физически в РБ, документ согласия с журналированием
