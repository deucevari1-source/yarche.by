# кондиционер.by — каталог и установка кондиционеров в Минске

**Сайт:** [konditsioner.by](https://konditsioner.by) · **Год:** 2026

Промо-сайт сервиса установки бытовых кондиционеров: 89 моделей сплит-систем с реальными ТТХ, калькулятор стоимости под ключ, лид-форма с Telegram-уведомлением и большая SEO-сетка по районам Минска, типам помещений и брендам.

## Стек

| Слой | Технология |
|---|---|
| Фреймворк | Next.js 14 (App Router), React 18, TypeScript |
| Стили | Tailwind CSS 3 |
| Картинки | next/image + WebP |
| Парсер прайса | Python 3 + openpyxl + Pillow |
| Хостинг | VPS, Беларусь · nginx + Let's Encrypt |

## Что внутри

- **~100 пререндеренных страниц** под SEO: каталог, бренды, районы Минска, типовые задачи, сравнения, блог
- **Автообновляемый каталог** — Python-скрипт тянет 89 моделей с фото и характеристиками прямо из xlsx-прайса дистрибьютора, без ручного ввода
- **Калькулятор стоимости** под ключ с учётом площади, типа стен и материалов — единая логика и на клиенте, и на сервере
- **Лид-форма** с уведомлениями в Telegram менеджерам
- **JSON-LD** на каждом типе страниц (Product, LocalBusiness, FAQPage, Article)
- В мае 2026 — миграция домена `air20.by` → `konditsioner.by` с сохранением SEO-сигналов
